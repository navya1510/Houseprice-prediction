# House Price Prediction Using Machine Learning

## Overview
A machine learning project that predicts house prices based on property features and housing attributes. The project includes data preprocessing, feature engineering, exploratory data analysis, and model evaluation.

## Technologies Used
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Scikit-learn

## Workflow
- Data Cleaning
- Exploratory Data Analysis (EDA)
- Feature Engineering
- Correlation Analysis
- Model Training
- Model Evaluation
- House Price Prediction
